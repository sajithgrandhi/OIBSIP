\# 🚗 Car Price Prediction using Machine Learning



\## 📌 Project Overview



This project was developed as part of the \*\*Oasis Infobyte Data Science Internship\*\*.



The objective is to predict car prices using Machine Learning based on different car features.



\---



\## 🎯 Objective



\- Analyze the car price dataset.

\- Perform data preprocessing.

\- Train a Machine Learning model.

\- Predict car prices.

\- Evaluate model performance.



\---



\## 🛠️ Technologies Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Scikit-learn



\---



\## 📂 Dataset



Dataset Used:



\*\*CarPrice\_Assignment.csv\*\*



The dataset contains features such as:



\- Car Name

\- Fuel Type

\- Aspiration

\- Horsepower

\- Engine Size

\- Car Width

\- Car Height

\- Drive Wheel

\- Price
